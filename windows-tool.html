What the tool does

CyberShield Hardener is a small GUI on top of PowerShell. It asks your permission before each action, shows you the exact command it will run, and logs everything it does so your IT person can review it later.

🛡
Windows Defender sweepVerifies real-time protection, signature freshness, and launches a quick scan.
🔥
Firewall auditConfirms all three firewall profiles are enabled and lists risky inbound rules.
🔑
BitLocker checkReports drive encryption status and offers to enable it on the OS volume.
📦
Update statusLists pending Windows updates and missing security patches.
👥
User account reviewShows local admins, enabled guest accounts, and weak password policies.
🧹
Startup cleanupLists unknown startup programs and scheduled tasks for easy review.
🌐
DNS & networkFlags suspicious DNS settings, proxy overrides, and unknown hosts file entries.
📜
Evidence bundleExports a signed .zip of logs you can send to an incident-response helpline.

Example commands it will run

Every command requires your click to confirm. Nothing runs silently in the background.

DefenderGet-MpComputerStatus | Select-Object AMServiceEnabled, RealTimeProtectionEnabled, AntivirusSignatureLastUpdated
FirewallGet-NetFirewallProfile | Select-Object Name, Enabled
BitLockerGet-BitLockerVolume | Select-Object MountPoint, VolumeStatus, ProtectionStatus
UpdatesGet-WindowsUpdateLog; Get-HotFix | Sort-Object InstalledOn -Descending | Select -First 10
AccountsGet-LocalUser | Where-Object Enabled -eq $true; Get-LocalGroupMember -Group "Administrators"
StartupGet-CimInstance Win32_StartupCommand | Select-Object Name, Command, Location, User
NetworkGet-DnsClientServerAddress; Get-NetIPConfiguration; Get-Content $env:windir\System32\drivers\etc\hosts
EvidenceCompress-Archive -Path $env:TEMP\cybershield-logs -DestinationPath $HOME\Desktop\CyberShield-Evidence.zip

Download & install

Download the installer. A signed .exe (~8 MB) will be published here soon.
Verify the signature. Right-click the file → Properties → Digital Signatures, and confirm it's signed by "CyberShield Project".
Run as administrator. Most hardening commands need elevated privileges.
Pick the checks you want. Each one shows the PowerShell it will run and asks for confirmation.
Export the evidence bundle. Attach it to your CyberShield action plan when asking an expert for help.

Safety & privacy

CyberShield Hardener is designed with NGO threat models in mind:

  • Runs entirely on your computer — no data is uploaded anywhere.
  • Open source so anyone (your tech-savvy volunteer, Access Now, a pen tester) can audit the code.
  • Every action is logged locally so changes can be reviewed or reverted.
  • Works offline, which matters if your Wi-Fi is still considered hostile.

CyberShield Hardener does not replace a professional incident response. If your NGO is under active attack, contact help@accessnow.org immediately.
